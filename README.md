# Nuxt Minimal Starter

Look at the [Nuxt documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.

## Technologies Used

- **Frontend Framework**: Vue.js 3 with TypeScript and Nuxt.js 4
- **Build Tool**: Vite
- **UI Components**: 
  - Element Plus
  - icomoon
  - Tailwind CSS

## Getting started

```bash
pnpm install 
```

## Start a Project

```bash
pnpm dev
```
